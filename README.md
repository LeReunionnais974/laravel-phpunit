## Installation

```bash
  cd laravel-phpunit
```

```bash
  composer install
```

```bash
  cp .env.example .env
```

```bash
  php artisan key:generate
```

```bash
  npm install
  npm run dev
```

```php
  php artisan migrate --seed
```

```php
  php artisan test
```
